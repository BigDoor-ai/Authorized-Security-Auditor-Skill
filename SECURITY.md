# Security Policy

## Reporting a vulnerability in this repository

Do not publish secrets, client data, or exploit details in a public issue.

Provide:

- Affected version or commit
- Description
- Reproduction using synthetic data
- Security impact
- Suggested remediation, when available

## Scope

This repository contains instructions and local utilities. It does not provide authorization to test third-party systems.

## Supported versions

Security fixes are maintained for the latest released version.
