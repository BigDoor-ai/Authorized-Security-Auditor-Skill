# Limitations

Common limitations that must be disclosed:

- No source-code access
- No infrastructure or cloud access
- No database administrative access
- No mobile build
- No test accounts for one or more roles
- Production-only environment
- Third-party systems excluded
- Rate limits or test-window restrictions
- Automated scanners not approved
- No social engineering, wireless, physical, or load testing
- Incomplete logs or backup restoration evidence
- Features unavailable during the test period
- Changes deployed during testing

Never convert missing evidence into a pass. Mark it `[Unverified]` or `[Blocked]`.
